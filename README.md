## Getting Started

To explore and run your portfolio locally, follow these simple steps:

```bash
npm install
npm run dev
# or
yarn install
yarn dev
# or
pnpm install
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to explore the portfolio.
